Help / Controls
KEYBOARD
• Arrows: Move / Turn
• Shift + Arrows: Slide
• Space / F: Fire
• Del / Backspace: Remove obstacle
• C: Cloak
• A: Appear
• M: Manual Play
• R: Random
MOUSE
• Left Click: Rotate CCW
• Right Click: Rotate CW
• Canvas Click: Place Player (when "Appear" is active)
TIPS
• Red flash = damage taken or hit obstacle
• 5 hits = destroyed
• Ammo refills only in Random mode via Reload button

Warning

Manually resizing the canvas will completely reset and clear the current game. Do you want to proceed?
